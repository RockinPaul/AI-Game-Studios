# Review Workflow

1. Code changes require review by the relevant department lead agent unless user-granted autonomy explicitly covers proceeding without that checkpoint.
2. Design changes require sign-off from `game-designer` and `creative-director` unless user-granted autonomy explicitly covers proceeding without that checkpoint.
3. Architecture changes require sign-off from `technical-director` unless user-granted autonomy explicitly covers proceeding without that checkpoint.
4. Cross-domain changes require sign-off from `producer` unless user-granted autonomy explicitly covers proceeding without that checkpoint.

When a user-granted autonomy instruction skips a review checkpoint, record that
the checkpoint was skipped by user instruction and summarize residual risk.
